# 🛡️ Security Policy

We take the security of VakyaLang seriously. If you believe you have found a vulnerability, please report it privately.

## Reporting a Vulnerability
- Please do **not** open a public issue for security bugs.
- Send a detailed report to the maintainer via private channels or the repository's security advisory tool.
- Include a summary of the issue, steps to reproduce, and any suggested fixes.

We aim to respond to security reports within 48 hours and provide a fix or mitigation strategy shortly thereafter.
