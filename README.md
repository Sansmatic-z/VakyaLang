**वाक् वै ब्रह्म** — *Speech is indeed the universal principle.*

**VakyaLang** is a Sanskrit-inspired symbolic programming language and computing framework that treats Sanskrit as a rigorous formal substrate for modern computing — rooted in Pāṇinian grammar and Nyāya logic.

### Core Components
- **Vāk (वाक्)**: General-purpose VM-based language with Devanagari syntax
- **Sanskrit Coder**: Symbolic reasoning engine using natural Sanskrit + Nyāya syllogisms

---

## 🚀 Quick Start

```bash
git clone https://github.com/Sansmatic-z/VakyaLang.git
cd VakyaLang
pip install -e .
```

```bash
vak examples/namaste.vak          # Run program
vak repl                         # Interactive REPL
sanskrit-coder                   # Symbolic engine
```

## 📂 Repository Structure
- `runtime/` — Core language runtime, VM, compiler, interpreter + stdlib
- `sanskrit_coder/` — Sanskrit-native math, logic & grammar engine
- `docs/` — Full specification, philosophy & research papers
- `examples/` — Ready-to-run programs (namaste, Fibonacci, OOP, etc.)
- `tests/` — Complete test suite
